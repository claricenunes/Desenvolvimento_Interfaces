# Desenvolvimento_Interfaces

Repositório criado para armazenar atividades, exercícios e projetos desenvolvidos na disciplina Desenvolvimento de Interfaces.

# Sobre a disciplina

A disciplina de Desenvolvimento de Interfaces tem como objetivo ensinar conceitos e práticas para criação de interfaces web, focando em usabilidade, estrutura e apresentação de páginas.

# 🛠 Tecnologias utilizadas

![My Skills](https://skillicons.dev/icons?i=html,css,js)


# 📂 Estrutura do repositório

O repositório está organizado por aulas e atividades.

Exemplo:

aula01/
aula02/

Cada pasta contém os arquivos desenvolvidos durante as aulas ou exercícios propostos.

# Objetivo

Praticar e aplicar conceitos de desenvolvimento web voltados para a construção de interfaces, incluindo:

Estruturação de páginas com HTML

Estilização com CSS

Organização de layout

Boas práticas de desenvolvimento front-end
